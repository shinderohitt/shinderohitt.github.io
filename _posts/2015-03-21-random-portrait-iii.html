Here it is! First portrait with charcoal pencils. All my assumptions were wrong. It's actually damn hard to get the shades right with these pencils compared to the graphite ones. Observation -- you need to be really, really precise with the pressure you apply with these. And so, it will take a lot of time to complete the same portrait with the same accuracy with charcoal pencils compared to graphite ones. Anyway, this thing is bad, but the first one, so should be spared.
